# 🎓 University ERP Login Page

This project is a **University ERP Login Interface** designed for students to securely log into the portal.  
The interface provides a clean and modern login UI with essential authentication fields such as **email, password, and user role selection**.

## 📸 Project Preview

This login page includes:

- University logo and header
- Login form interface
- Email input field
- Password input field
- Remember me checkbox
- Forgot password option
- Login button
- Campus background image
